### What This Project Covers

This final project is a **fully-functional Todo App** that combines everything from the course:

| Feature | Concepts Used |
|---------|--------------|
| `Todo` class with `#` private fields | Day 15 — ES6 Classes |
| `TodoStore` with Observer pattern | Day 29 — Design Patterns |
| DOM rendering + event delegation | Day 12, 13 — DOM |
| `localStorage` persistence | Day 22 — LocalStorage |
| Form handling | Day 14 — Forms |
| Debounced search with regex | Day 23, 25 — Regex + HOFs |
| `Map` for O(1) todo lookup | Day 27 — Map & Set |
| Error handling throughout | Day 20 — Error Handling |
| Closures for private state | Day 24 — Closures |
| ES6 classes, spread, destructuring | Days 10, 11, 15 |

### Key Features

- ✅ Add, edit, delete, toggle todos
- ✅ Priority levels (high/medium/low)
- ✅ Due dates with overdue detection
- ✅ Tags system
- ✅ Filter by all/pending/completed/overdue
- ✅ Sort by date/priority/alphabet
- ✅ Live search with debounce
- ✅ Stats dashboard
- ✅ Persistent storage (localStorage)
- ✅ Immutable state updates

### How to Run

Open `index.html` in a browser — no build tools needed!
