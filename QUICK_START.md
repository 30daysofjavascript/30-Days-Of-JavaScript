# ⚡ Quick Start Guide

## Running JavaScript Files

### In the Browser Console (Easiest)
1. Open Chrome/Firefox
2. Press `F12` to open DevTools
3. Go to the **Console** tab
4. Copy/paste code from any `index.js` file

### In Node.js (No Browser Needed)
```bash
# Install Node.js from https://nodejs.org first
node Day-01-Variables-DataTypes/index.js
node Day-06-Functions/index.js
```

### For DOM Days (12, 13, 14, 30)
Open the `index.html` file directly in your browser — it will load the `index.js` automatically.

## Recommended Order
Follow days 1–30 in order. Each day builds on the previous.

## How to Get the Most Out of This Course
1. **Read the code comments** — every concept is explained inline
2. **Type the code yourself** — don't just read it
3. **Complete the exercises** at the bottom of each file
4. **Experiment** — change values, break things, see what happens
5. **Build the Day 30 project** to cement everything

## Tools
- **VS Code** — best editor for JS: https://code.visualstudio.com
- **Node.js** — run JS outside browser: https://nodejs.org
- **Chrome DevTools** — built-in debugger (F12)

Happy coding! 🚀
